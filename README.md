## 📌 Project Overview

The **IPL Data Visualization** project focuses on analyzing historical Indian Premier League (IPL) cricket data to uncover meaningful trends, team performances, and player insights.

Using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**, this project transforms raw match statistics into visually engaging charts and dashboards that help understand the evolution of IPL over the years.

---

## ✨ Features

### 🔹 Data Cleaning & Preprocessing
- Handling missing values

- Preparing datasets for analysis

## 📊 Dataset

### IPL.csv

The dataset contains comprehensive IPL match information, including:

| Column Category | Description |
|---------------|-------------|
| Teams | Participating teams |
| Venue | Match venue |
| Toss Details | Toss winner and decision |
| Match Results | Winning team and margins |
| Awards | Player of the Match |
| Match Metadata | Season and match details |
